Project 1: Twitter

Problems to solve

Twitter has changed the journalistic landscape.

The majority of Australian journalists and news outlets use Twitter daily, for sourcing and distributing news.

When there's breaking news, how does a journalist or a member of the public sift through the noise in their feed to get the facts?

#breakingnews Tweets

Solution

Journalists and news outlets are verified when they sign up.

Users can easily switch between a feed for 'News' and a feed for 'All'.

  • Replies? (many)
  • Location?

Views

Log in/Sign up

Allows the user to sign up or log in if they already have an account.

News

Displays a timeline of all tweets from a journalist or news organisation that you are following. This would be the default view.

All

Displays a timeline of all tweets from everyone you are following.

New tweet

Allows the user to create a new tweet.

Tweet

Shows single tweet on page.

User

Shows single user on page.

User tweets

Displays all of the users tweets.

Following

Displays everyone who the user is following.

Followers

Displays the users followers.

Mentions

Displays the users mentions.
